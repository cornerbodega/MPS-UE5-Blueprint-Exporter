## [1.0.0] - 2025-01-09

### Added
- C++ plugin for blueprint graph extraction
- Python orchestration script (blueprint_watcher.py)
- Markdown generation from JSON exports
- Full blueprint data extraction (nodes, pins, connections)
- Variables, functions, and components extraction
- Dependency tracking
- Claude Code integration instructions
- Installation guides for Mac and Windows
- MIT License
- Comprehensive README with examples
- Setup instructions and troubleshooting guide

### Features
- Export blueprints to JSON format
- Export blueprints to Markdown format
- Index generation for all blueprints
- Support for UE5.0+
- Works with Blueprint-only projects (adds C++ support)
